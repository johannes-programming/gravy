Overview

Calculate the GRAVY score of an amino acid sequence.

Usage

CLI


python3 -m gravy CYIQNCPLG
# output: 0.33333
# the formatting can be changed with the option '--format'

Python


import gravy
x = gravy.score("CYIQNCPLG")
y = format(x, ".5f")
print(y)
# output: 0.33333
# the same format as the default in the CLI
